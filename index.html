o Project title   JavaScript program to calculate multiplication of two numbers using Karatsuba algorithm.


o Input specifications   Two integers to be Multiplied.


o Objective   To multiply two numbers of size n , in the minimum possible time complexity as prescribed by the Karatsuba Algorithm.


o Pseudocode


  1. procedure karatsuba(num1, num2)
  2.   if (num1 < 10) or (num2 < 10)
  3.     return num1*num2;

  5.   else
  6.   /*if numbers have different bit size, 0 will be added to equal them*/
  7.   /*if any number is odd, next even digit will be used*/
  8.   n = max(size(num1),size(num2)
  9.   low1, low2 = lower half of num1, num2
  10.   high1, high2 = higher half of num1, num2
  11.   z0 = karatsuba(low1,low2)
  12.   z1 = karatsuba((low1+high1),(low2+high2))
  13.   z2 = karatsuba(high1,high2)
  14.   c1 = pow(2,n)-pow(2,n/2);
  15.   c2 = pow(2,n/2);
  16.   c3 = 1-pow(2,n/2);
  17.   return c1*z2+c2*z1+c3*z0
